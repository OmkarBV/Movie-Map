# Movie-Map

- Technologies: Java, Spring MVC, Hibernate, JSP, JavaScript
- Database: MYSQL
- Overview: Movie Map a Spring MVC and Hibernate-based web application with two modules - admin and user. The admin module allows administrators to perform CRUD operations, while the user module enables users to search, sort, and rate movies based on various criteria. The application uses a MySQL database to store movie data, making it a user-friendly platform for movie enthusiasts.
